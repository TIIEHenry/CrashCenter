# NeverCrash — Project Instructions

## Workflow Preference

**Do NOT use EnterPlanMode.** For any task requiring planning or multi-step exploration, use the `Agent` tool (sub-agents) instead. Sub-agents keep the main context clean and allow parallel work.

- Use `Explore` agents for codebase research
- Use `Plan` agents for implementation design
- Use `general-purpose` agents for multi-step tasks
